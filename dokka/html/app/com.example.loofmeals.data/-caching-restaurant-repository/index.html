CachingRestaurantRepository

class CachingRestaurantRepository(restaurantApiService: RestaurantApiService, restaurantDao: RestaurantDao) : IRestaurantRepository

Implementation of the RestaurantRepository interface that caches the restaurant data.

This class fetches the restaurant data from the network and stores it in a local database. When fetching the restaurant data, it first tries to fetch it from the local database. If the local database is empty, it fetches the data from the network and stores it in the local database.
