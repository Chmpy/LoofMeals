NavigationDrawerSmall

@Composable
fun NavigationDrawerSmall(navigationType: NavigationType, items: List<NavigationItem>, drawerState: DrawerState, scope: CoroutineScope, navController: NavHostController, goToOverview: () -> Unit, goToFavorite: () -> Unit, goToAbout: () -> Unit, goToMap: () -> Unit)

Composable function that creates a small navigation drawer using the Material 3 design components, incorporating a ModalNavigationDrawer and ModalDrawerSheet. The drawer displays a list of NavigationItem items with icons, labels, and optional badge counts. This version is suitable for smaller screens or contexts where a full-width drawer is not ideal, such as mobile or tablet-sized layouts.

Parameters

navigationType

The type of navigation, used in the RootLayout for configuring the interaction with the content of the screen.

items

List of NavigationItem representing each navigation destination in the drawer.

drawerState

DrawerState representing the state of the modal navigation drawer.

scope

CoroutineScope used for managing coroutine lifecycles.

navController

NavHostController used for navigation between destinations.

goToOverview

Callback function to navigate to the Overview destination.

goToFavorite

Callback function to navigate to the Favorites destination.

goToAbout

Callback function to navigate to the About destination.

goToMap

Callback function to navigate to the Map destination.
